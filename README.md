# 🤖 Sistema Multi-Agente: IA Tutora de Testes Unitários

Este projeto é um ambiente de aprendizado interativo focado em Testes Unitários com Java (JUnit 5 + Mockito).

## 🚀 Como Inicializar o Treinamento

Para começar sua jornada, basta enviar a mensagem:

**"Quero aprender Testes Unitários"** ou **"Iniciar Treinamento"**

## 📂 Estrutura do Projeto

*   `workspace/exercicios/`: Contém os desafios práticos (Exercícios 1 a 5).
*   `workspace/analises/`: Onde o "Inspetor de Código" deposita os feedbacks das suas soluções.
*   `agents/`: Definições das personas que te guiarão (Mestre, Professor, Arquiteto, Inspetor).

## 🎓 Metodologia

O treinamento é conduzido por uma orquestra de IAs especialistas:
1.  **Mestre dos Testes**: Orquestra sua jornada e avalia seu progresso.
2.  **Professor JUnit**: Ensina os conceitos teóricos quando necessário.
3.  **Arquiteto de Desafios**: Apresenta os exercícios práticos.
4.  **Inspetor de Código**: Avalia tecnicamente suas submissões JUnit 5.

Este sistema utiliza práticas de *Clean Code* e o padrão de testes *AAA (Arrange, Act, Assert)*.
